Help

DoIt! is a simple and easy to use task management application. To get started, just type your command into the command box in the main window. The following commands are accepted:

  • help
  • add, edit, postpone
  • list, sort,, search, refresh
  • done, undone
  • delete
  • undo
  • exit

Click on the command to see more details about the command. Or simply type the command word into the command box, and you will be able to see how to use it. Type "help on" if you have previously turned the command help off!

Or learn more about the user interface.

DoIt! supports several types of tasks:

  • Floating tasks: Tasks which have no time component.
  • Deadline tasks: Tasks which have one time as deadline.
  • Timed tasks: Tasks which have a start and end time.

User Interface

DoIt! user interface consist of forms: the agenda view and the list view.

Agenda View

The agenda view gives you an at-a-glance view of your tasks organised by date. It consists of two panes: the left pane shows your tasks that have a date (deadline and timed tasks) while the right pane shows your tasks with no date (floating tasks).

The calendar allows you to quickly jump to your tasks on a particular day. Days that you have tasks are bolded.

List View

The list view lets you see all your tasks in a spreadsheet-like form. You can also perform actions like editing by double-clicking on the respective cell, as well as deleting your tasks by selecting the respective rows you want to delete and then pressing the "delete" button.

Toolbar

The toolbar at the top of DoIt's main window contains the following buttons:

Home
Resets any search or list filters. Also (optionally) jumps to the default view (Agenda or List view) and jumps to today's tasks. Customize this in the settings.
Undo
Undo the last performed action.
Agenda View/List View
Switch to the selected view.
Preferences
Shows the preferences dialog.
Help
Shows this help dialog.

Command Box

Use the command box right at the bottom of the window to perform all sorts of operations on your tasks list. Type in the command you want and press Enter to execute the command. The status of the command will be shown in the space below.

Quick Add Box

DoIt! has a handy quick add box. Use the system tray icon to open the quick add box, or use the keyboard shortcut (default Windows key + A, customizable in perferences).

Commands

Add

Adds a task to database

Usage/Examples :

add read a book
add school camp from 3pm today to 5pm tomorrow
a hand in homework by Monday 6pm (a is a shortcut for add)

If DoIt misinterprets the task name as the time, put inverted commas around "task name":
add "read book from page 1000 to 1100" by Friday.

Edit

Edits the selected task

Usage/Examples :

edit [index] [parameter] [new value] ...
edit 1 -name go home (Change name of task at index 1 to "go home")
edit 10 -deadline today 6pm (Change deadline of task 10 to 6pm today. Will convert to deadline task if not previously.)
edit 2 -start 3pm 5 sept -end 4pm 6 sept (Change start and end times of timed tasks)

To maximise DoIT's chances in correctly updating your task, try to change one parameter at a time.

Postpone

Postpones a task

Usage/Examples :

postpone [index] [time parameter]
postpone 1 3 hours (postpones task at index 1 by 3 hours)

This command has no effect on floating tasks. Start and End times for timed tasks will shift together.

List

Shows the list of tasks in your schedule

Usage/Examples :

list [parameter(s)]
list today (Tasks that happen today)
ls undone floating (Floating tasks that are incomplete)
l overdue (Tasks that are before the current moment and are incomplete)

Possible parameters: "done", "undone", "timed", "deadline", "floating", "today", "tomorrow", "overdue"

Sort

Sorts the current list of tasks

Usage/Examples :

sort [parameter] "descending"
sort type (Order of deadline, timed, floating)
sort done (Undone tasks come first)
sort start (By start time/deadline)
sort end (By end time/deadline)
sort name (Alphabetical order)

Add "descending" or "reverse" parameter to sort by descending order instead.
Only up to 1 parameter except "descending" is allowed. Default is sort by start/deadline in ascending order.
Sort works best in List view. Sort will be maintained each time the view is updated

Search

Search for task(s) based on keywords

Usage/Examples :

search "keyword1" "keyword2" ...
search birthday (search for tasks with "birthday" in description)
search birthday family (search for tasks with "birthday" and "family" in description)

The keyword search is case-insensitive. Time range search is not supported yet.

Refresh

Refreshes your list based on the latest list or search parameters.

Usage/Examples :

undo

This command has no effect in the graphical interface.

Done

Marks the task specified by index as done.

Usage/Examples :

done [index]

Obtain the "index" number from the list. "index" will be based on last shown list.

Undone

Marks the task specified by index as undone.

Usage/Examples :

undone [index]

Obtain the "index" number from the list. "index" will be based on last shown list.

Delete

Deletes task(s) from your schedule

Usage/Examples :

delete [index] (Deletes the task at index)
delete done (Delete all completed tasks)
del over (Delete all tasks before this moment)
d all (Deletes EACH AND EVERY task in your schedule)

Undo

Undos the last change you made to the database.

Usage/Examples :

undo

DoIT places no limit on the number of undo steps

Exit

Exits DoIT

Usage/Examples :

exit
quit

All undo steps will be lost when DoIT quits.
