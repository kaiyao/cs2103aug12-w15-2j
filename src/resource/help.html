Help

DoIt! is a simple and easy to use task management application. To get started, just type your command into the command box in the main window. The following commands are accepted:

  • help
  • add, edit, postpone
  • list, sort, search, refresh
  • done, undone
  • delete
  • undo
  • exit

Click on the commands to learn more about them! Alternatively, just type the command word into the command box, and you will be able to see how to use it.

DoIt! supports several types of tasks:

  • Floating tasks: Tasks which have no time component.
  • Deadline tasks: Tasks which have one time as deadline.
  • Timed tasks: Tasks which have a start and end time.
